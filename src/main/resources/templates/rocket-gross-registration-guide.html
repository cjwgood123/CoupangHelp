로켓그로스 상품등록 A to Z

등록 → 노출 → 입고 통과 → 매출 흐름이 한 번에 잡히는 정답 루트

로켓그로스를 시작하면 대부분 여기서 갈립니다. "등록은 했는데 안 팔리는 사람" vs "등록하자마자 반응 오는 사람." 차이는 복잡한 게 아니라 딱 몇 가지 핵심을 지켰냐 안 지켰냐입니다. 지금부터는 진짜로 초보가 따라만 해도 실패 확률을 확 줄이는 등록 방식을 단계별로 풀어봅니다.

1 초보가 제일 많이 빠지는 함정: 아이템링킹(카탈로그 매칭)

로켓그로스 등록할 때 가장 쉽게 보이는 선택지가 "카탈로그 매칭"입니다. 이미 쿠팡에 잘 팔리는 상품을 찾아서 링크 붙여 등록하면 끝이니까요.

근데 이 방식은 "장사"가 아니라 "가격전쟁 참여"가 되어버립니다.

왜 아이템링킹이 위험한가?
  • 누구든 너랑 똑같이 들어올 수 있음
  • 더 싸게 파는 사람이 1명만 등장하면 → 너는 노출에서 밀리고 → 매출은 그대로 도둑 맞음
  • 결국 남는 건 "최저가 싸움 + 마진 붕괴 + 의미 없는 노동"

즉, 아이템링킹은 "지금은 편해 보여도 미래 매출을 버리는 선택"입니다.

✅ 그래서 로켓그로스는 무조건

"내 상품명 / 내 옵션 / 내 이미지로 독자 등록"이 정석입니다.

2 상품명(제목)이 매출의 절반이다 — 모바일 가독성 기준

쿠팡은 PC보다 모바일 유입이 훨씬 크고, 고객은 스크롤 내려가면서 1초 만에 클릭할지 말지를 결정합니다. 그래서 상품명은 이렇게 생각하면 됩니다:

핵심 원칙

"사람이 핸드폰으로 봤을 때 한 번에 이해되는가?"

브랜드명 맨 앞에 필수

  • 쿠팡에 등록된 브랜드여야 함
  • "노브랜드/무브랜드" 같은 표현 불가

상품 정체성은 짧고 정확하게

  • 고객이 딱 보는 순간 "뭐 파는지" 알아야 함
  • 옵션(색/수량/단수)은 상품명에 넣지 마
  • 옵션은 옵션칸에
  • 제목은 "상품 자체 설명"에 집중

📌 좋은 상품명 구조

[브랜드] + [상품명] + [핵심 특징]

예시:

  • 돈버는하마 캠핑 식기건조대 접이식
  • 돈버는하마 차량용 선풍기 무선 대용량
  • 돈버는하마 실리콘 접이식 대야 대형

👉 "그래서 이게 뭔데? + 왜 필요한데?" 이 두 가지가 제목 안에서 끝나야 클릭이 나옵니다.

3 카테고리 선택이 '옵션 구조'를 강제로 정한다

초보가 놓치는 포인트가 바로 이것입니다. 카테고리 = 그냥 분류가 아니라, "옵션 설계도"입니다.

카테고리 잘못 잡으면 생기는 문제
  • 옵션 구조가 엉망이 됨
  • 필수 옵션 누락/불일치 발생
  • 결과적으로 등록은 되는데 노출이 안 되거나, 입고에서 반려

✅ 카테고리 고르는 가장 확실한 방법

  1. 쿠팡에서 내 상품 키워드로 검색
  2. 상위 판매상품 3~5개 클릭
  3. 그 사람들이 쓰는 카테고리 확인
  4. 똑같이 선택

✅ "내가 편한 카테고리"가 아니라 "쿠팡이 정해둔 집"에 들어가야 합니다.

4 옵션은 '내 마음'이 아니라 '카테고리 룰'대로

카테고리를 정하면 쿠팡이 요구하는 필수 옵션이 뜹니다. 예: 색상 / 단수(티어) / 수량

✅ 옵션 세팅 원칙

  • "설정함(옵션 있음)"으로 맞춰라 (단일상품이어도 그래야 함)
  • 옵션 이름은 명확하고 단순하게

예시:

  • 화이트 / 3단 / 1개
  • 베이지 / 3단 / 1개

옵션을 억지로 빼거나 바꾸면 노출룰·입고룰에서 튕기기 딱 좋습니다.

5 가격: 판매가 + 정상가 = "할인 구조"를 만들어라

쿠팡은 기본적으로 정상가 대비 판매가가 낮을수록 구매 전환이 올라갑니다.

✅ 가격 세팅 룰

  • 판매가 먼저 결정
  • 정상가는 판매가보다 높게 → 고객에게 "할인 중" 느낌 제공
  • 90% 이상 과할인 금지 (쿠팡 정책상 제한됨)

계산 기준(실전)

판매가 15,900원이라면

정상가 = 15,900 / 0.7 ≈ 22,700원

→ 약 30% 할인처럼 보이게 만드는 구조.

이건 "심리 할인"이 아니라 쿠팡에서 기본적으로 잘 먹히는 가격 연출 공식입니다.

6 재고/배송 채널 전략: 등록할 땐 둘 다 열어둬라

로켓그로스 등록할 때 "판매자배송(윙)" + "로켓그로스" 둘 다 체크하는 게 안정적입니다.

왜 둘 다 체크해야 하나?

로켓그로스 물량이 먼저 다 팔릴 수 있음. 그 순간 주문이 끊기면 → 랭킹/노출이 눌림 → 다시 올리기 어려움

✅ 전략

그로스만 운영할 거면 → 윙 재고 0으로 설정하지만, 그래도 채널은 열어두는 게 안전장치입니다.

📌 결론: 판매 흐름이 끊기지 않게 "보험" 하나 걸어두는 개념입니다.

7 이미지가 진짜 "입고 합격/반려"를 결정한다 (최중요)

로켓그로스를 하면서 제일 비싼 실수는 이미지입니다. 입고 검사 때 물류센터 직원이 하는 일은 간단합니다:

"이미지 = 실물 일치 여부만 체크"

❌ 반려 나오는 대표 케이스
  • 옵션 이미지가 색상/형태랑 다름 (화이트 사진인데 베이지 보내면 바로 반려)
  • 구성품 개수가 사진과 다름 (사진엔 6개인데 실물 5개면 반려)
  • 연출컷에 "구성품 아닌 물건"이 나옴 → 물류센터가 "누락"으로 판단

✅ 이미지 세팅 정석

  • 옵션별 대표이미지는 무조건 따로
  • 실물 그대로
  • 불필요한 텍스트/치수/중국어 제거
  • 밝기/대비 살려서 모바일에서 제품이 또렷하게 튀게

👉 이미지에서 문제 나면 입고비/반송비/시간이 전부 네 비용이 됩니다. 그래서 여기만큼은 "과할 정도로 정확하게."

8 검색키워드 20개 + 필터 풀 세팅 = 노출 통로 확장

쿠팡은 검색/필터 노출이 대박 큽니다. 태그 20개를 "안 채우면 내 상품을 스스로 숨기는 수준"입니다.

✅ 키워드 세팅 방법

  1. 메인 키워드 검색
  2. 아이템스카우트 등으로 연관어 수집
  3. 상위 노출 키워드 복사
  4. 20칸 꽉 채우기

예:

  • 캠핑 식기 건조대
  • 접이식 식기 건조대
  • 캠핑 설거지 건조대
  • 캠핑 수납 건조대
  • 3단 식기건조

고객이 실제 쓸 말로 쌓는 게 핵심입니다.

✅ 필터도 최대한 많이

접이식/단수/사이즈/커버 유무 등 쓸 수 있는 필터는 다 체크해야 노출 "루트"가 늘어납니다.

9 최종 등록 흐름 한 장으로 정리

로켓그로스 상품등록 승부는 딱 이 흐름

아이템링킹 NO → 브랜드+모바일가독 제목 → 정확한 카테고리 → 옵션 구조 그대로 → 판매가/정상가 할인 세팅 → 재고 채널 안전장치 → 옵션별 실사 이미지 → 키워드 20개 풀 + 필터 풀세팅

이대로 하면 "반려 없이 입고되고, 노출되고, 팔리기 시작하는" 등록이 됩니다.

마무리

로켓그로스는 결국 "등록 방식이 곧 매출 구조"입니다.

급하게 따라 붙는 아이템링킹보다 내 상품을 제대로 등록하고, 옵션/이미지/키워드로 승부하는 사람만 오래 갑니다.

오늘 내용대로 3개만 등록해보세요. 등록 방식 하나 바꿨을 뿐인데 노출이 달라지고 판매가 시작되는 경험을 하게 될 겁니다.
