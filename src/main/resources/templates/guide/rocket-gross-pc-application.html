로켓그로스 PC 입고 신청 방법

쿠팡윙에서 입고 신청하는 단계별 가이드

로켓그로스 입고는 쿠팡윙(판매자센터)에서 PC로 신청합니다. 입고 신청부터 문서 출력까지 전체 과정을 단계별로 상세히 안내합니다.

✅ PC 입고 신청은 쿠팡윙에서만 가능

로켓그로스 입고 신청은 모바일이 아닌 PC(데스크톱)에서만 가능합니다. 쿠팡윙(판매자센터)에 로그인하여 입고 신청을 진행합니다.

1. 쿠팡윙 접속 및 로그인

먼저 쿠팡윙(판매자센터)에 접속하여 로그인합니다.

접속 방법

  1. 쿠팡윙(wing.coupang.com) 접속
  2. 판매자 계정으로 로그인
  3. 메뉴에서 "로켓그로스" 선택

필수 준비사항

  • 판매자 계정 로그인 정보
  • PC 환경 (모바일 불가)
  • 인터넷 연결
  • 프린터 준비 (문서 출력용)

2. 입고 신청 절차

쿠팡윙에서 입고 신청을 진행하는 단계별 과정입니다.

입고 신청 단계

  1. 로켓그로스 메뉴 선택: 쿠팡윙 메뉴에서 "로켓그로스" 클릭
  2. 입고 신청 클릭: "입고 신청" 또는 "신규 입고 신청" 버튼 클릭
  3. 상품 선택: 입고할 상품 선택 (상품번호 또는 상품명으로 검색)
  4. 수량 입력: 입고할 수량 입력
  5. 물류센터 선택: 입고할 물류센터 선택
  6. 배송 방식 선택: 배송 방식 선택 (직배송/택배 등)
  7. 신청 완료: 모든 정보 확인 후 신청 완료
💡 입고 신청 팁

입고 신청 전에 상품이 개별 포장되어 있고, 라벨이 부착되어 있는지 확인하세요. 신청 후에는 수정이 어려울 수 있으므로 신중하게 입력하세요.

3. 문서 출력 (필수)

입고 신청이 완료되면 반드시 출력해야 하는 문서들이 있습니다.

출력해야 할 문서

  • 부착 문서: 박스 겉면에 붙이는 문서
  • 동봉 문서: 박스 안에 넣는 문서 (옵션 혼합 시 필수)
  • 입고 라벨: 상품에 부착할 바코드 라벨

문서 출력 방법

  1. 입고 신청 완료 후 "문서 출력" 버튼 클릭
  2. PDF 또는 인쇄 버튼으로 출력
  3. A4 용지에 출력 (가정용 프린터 사용)
  4. 문서를 확인하고 잘라서 사용
⚠️ 중요

부착 문서는 반드시 박스 겉면에 붙여야 합니다. 문서가 없거나 잘못 붙이면 입고가 지연되거나 회송될 수 있습니다.

4. 입고 신청 시 주의사항

입고 신청 전 확인

  • 상품이 개별 포장되었는가?
  • 라벨이 준비되었는가?
  • 포장재와 박스가 준비되었는가?
  • 물류센터와 배송 방식을 정했는가?

입고 신청 후 확인

  • 문서가 정상적으로 출력되었는가?
  • 문서의 정보가 정확한가?
  • 입고 신청 정보가 올바른가?
  • 수정이 필요한 부분이 없는가?
💡 실수 방지 팁

입고 신청 전에 모든 준비가 완료되었는지 체크리스트로 확인하세요. 신청 후 수정은 어려울 수 있으므로 신중하게 진행하세요.

마무리

PC 입고 신청은 로켓그로스 입고의 첫 단계입니다.

쿠팡윙에서 신중하게 신청하고, 문서를 정확히 출력하여 안전하게 입고하세요.
