# 구글·네이버 검색 노출을 위한 SEO 가이드

## 현재 완료된 작업

### 1. 메타 태그 최적화
- ✅ Title/Description에 **헬쿠**, **쿠팡 매출**, **스마트스토어 무료 매출**, **무료 매출**, **로켓그로스 엑셀** 키워드 반영
- ✅ 메인(index): "헬쿠는 … 쿠팡 매출, 스마트스토어 무료 매출, 로켓그로스 엑셀·매출 일지" 노출
- ✅ 매출일지(margin-tracker): 쿠팡 매출·스마트스토어 무료 매출·로켓그로스 엑셀 무료 관리 타이틀/설명
- ✅ 로켓그로스 가이드: "로켓그로스 엑셀" 타이틀·키워드 추가
- ✅ Open Graph / Twitter Card 최적화, canonical·og:url 설정

### 2. 구조화된 데이터 (JSON-LD)
- ✅ WebSite 스키마(메인)에 헬쿠, 쿠팡 매출, 무료 매출, 로켓그로스 엑셀 키워드
- ✅ 매출일지 페이지 WebApplication 스키마 (무료 매출 관리 도구)

### 3. 사이트맵 및 robots.txt
- ✅ sitemap.xml에 **매출일지(/margin-tracker)** 추가, priority 0.95
- ✅ 보드 URL을 새 구조로 반영: `/board/100`, `/board/500`, `/board/1000` (기존 `/2025-11/all/` 대체)
- ✅ sitemap lastmod 전면 갱신: **2026-02-07**
- ✅ robots.txt: `/margin-tracker`, `/board/` 명시적 Allow, 구글·네이버 봇 공통 적용

## 구글 서치 콘솔 등록 필수!

### 1단계: 구글 서치 콘솔 접속
1. https://search.google.com/search-console 접속
2. Google 계정으로 로그인

### 2단계: 속성 추가
1. "속성 추가" 클릭
2. URL 접두어 방식 선택
3. `https://helpcoupang.com` 입력
4. 소유권 확인 (HTML 파일 업로드 또는 메타 태그)

### 3단계: 사이트맵 제출
1. 왼쪽 메뉴에서 "사이트맵" 클릭
2. `https://helpcoupang.com/sitemap.xml` 입력
3. "제출" 클릭

### 4단계: 색인 생성 요청
1. 왼쪽 메뉴에서 "URL 검사" 클릭
2. 메인 페이지 URL 입력: `https://helpcoupang.com`
3. "색인 생성 요청" 클릭
